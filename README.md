# express_crud 

# Installation

*Git Clone*
  ```
    git clone https://github.com/mohiuddin007/express_crud.git
  ```  
*Install all dependency*
  ```
    yarn
  ```
  or
  ```
   npm install
  ```
*Project run locally*
  ```
    yarn start:dev 
  ```
  or
  ```
    npm run start:dev
  ```
*Build*
   ```
     yarn build 
   ```
   or
   ```
   npm run build
   ```
*Run in production*
   ```
    yarn start:prod 
   ``` 
   or
   ```
    npm run start:prod
   ```
